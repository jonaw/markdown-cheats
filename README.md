# Lists
## Ordered
1. Item 1
	* Sub item
	* Sub item
		* Sub-sub item
2. Item 2
3. Item 3
```
1. Item 1
        * Sub item
        * Sub item
                * Sub-sub item
2. Item 2
3. Item 3
```
## Unordered
* Unordered item
	* Unordered sub item
* Unordered item
	1. Ordered sub item
* Unordered item
```
* Unordered item
        *Unordered sub item
* Unordered item
        1. Ordered sub item
* Unordered item

```
